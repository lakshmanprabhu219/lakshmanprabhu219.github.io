Pneumonia Detection

Using the Image Dataset available from Kaggle, I was able to develop a machine learning model that intelligently classifies if a person is affected by pneumonia or not, given the images of lung x-rays. The model had an accuracy of 0.95 on the training set and over 0.85 on the validation set. I used Google's brilliant Tensorflow API for this purpose.
